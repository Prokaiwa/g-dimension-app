How to apply in Supabase

  1. Go to Dashboard → Authentication → Email Templates
  2. Select the template type matching the tab above
  3. Paste the HTML into the Message body field (HTML mode)
  4. Set Subject line — suggestions below each template
  5. Set From name: G-Dimension and From address: hi@gdimension.app
  6. Save and send a test email to verify rendering
