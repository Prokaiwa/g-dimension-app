Template Variables — What each uses

Invite User — uses {{ .ConfirmationURL }} — the link that creates the account and signs the user in.
Magic Link — uses {{ .ConfirmationURL }} — a one-time sign-in link, no password required.
Reauthentication — uses {{ .Token }} — a numeric OTP code the user types into the app. No link button needed.

How to apply in Supabase

  1. Go to Dashboard → Authentication → Email Templates
  2. Select the template type matching the tab above
  3. Paste the HTML into the Message body field
  4. Set the Subject line — shown as a comment at the top of the code panel
  5. From name: G-Dimension  ·  From address: hi@gdimension.app
  6. Save and send a test to verify rendering
