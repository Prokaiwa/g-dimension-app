# Audio license proof

Drop your Pixabay download/license screenshots (PNG/JPG) in this folder, one
per audio file you ship. This keeps proof-of-license version-controlled and in
one place.

**Why here and not `public/audio/`:** anything in `public/` is served on the
live site and included in every build, so screenshots placed there would be
publicly downloadable (and can expose your Pixabay account name/email). This
`docs/` folder is part of the repo but is NOT deployed.

Suggested naming so each shot maps to its track:

| Audio file (`public/audio/…`) | License screenshot here |
|---|---|
| `confirm.mp3` | `confirm-pixabay.png` |
| `music.mp3` | `music-pixabay.png` |

Pixabay content is royalty-free, commercial-use OK, no attribution required —
the screenshot is just belt-and-suspenders proof of where it came from.
